# Food Exploration Workflow

Food exploration is the first reference workflow for OpenMelon.

Example request:

```text
I want to publish a realistic social post about eating beef noodles after work.
```

OpenMelon stages:

1. identify food exploration intent,
2. select the `food-street-realism` Skill-Plus package,
3. compile it for `openmelon` and the target image model,
4. generate a concrete visual prompt,
5. generate image assets,
6. review realism and artifact signals,
7. attach labels and provenance,
8. write feedback into project memory.
